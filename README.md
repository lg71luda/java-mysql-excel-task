# Анализ чисел с сохранением в MySQL и экспорт в Excel

Программа на Java, реализующая интерактивное консольное меню для:
- Проверки чисел на целостность и чётность,
- Сохранения результатов в базу данных MySQL,
- Экспорта данных из MySQL в файл Excel (.xlsx).

## Функционал

1. **Вывести все таблицы из MySQL**
2. **Создать таблицу в MySQL** (`number_analysis`)
3. **Проверить числа на целостность и чётность**  
   Пример ввода: `42 13.5 abc -8 0`
4. **Сохранить данные из MySQL в Excel** и вывести на экран

## Требования

- **Java 17 или выше** (проверено на Java 25)
- **MySQL Server** (локальный или удалённый)
- **Библиотеки** (включены в папку `lib`):
  - MySQL Connector/J 8.0.33
  - Apache POI 5.2.4
  - Apache Commons IO, Compress, Collections4
  - Log4j 2.23.1
